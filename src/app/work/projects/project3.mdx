## Overview

The Ticketing and Helpdesk System was created to provide seamless support for users of the Public Procurement Information Portal. It handles user queries, technical issues, and administrative requests through an organized and automated workflow.

## Key Features

- **Ticket Lifecycle Management:** Allows users to create, track, and resolve support tickets.  
- **Email & Notification Integration:** Keeps users updated on issue progress.  
- **Analytics Dashboard:** Visualizes support trends to improve service delivery.  
- **Role-Based Access Control:** Distinguishes permissions for admins, support agents, and users.

## Technologies Used

- **Stack:** Laravel, MySQL, Faveo Helpdesk API, Vue.js  
- **Automation:** Cron Jobs, Webhooks  
- **Hosting:** Apache Server, cPanel Deployment

## Outcome

The system streamlined communication between citizens and the procurement authority, cutting response times by over 60% and increasing satisfaction among platform users.
